# GUDI Web App

GUDI is a web application for intelligent digitalization and automation for SMEs.  
This repository contains the base architecture of the project, including frontend, backend, database setup, and technical documentation.

## Tech stack

- Frontend: React + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL
- ORM: Prisma
